一：初始化项目：
	1、初始化仓库： git init

	2、暂存文件：git add readme.txt（把它放到准备提交的队列里）

	3、提交文件：git commit -m "这里写本次提交的说明"

二：远程连接仓库
	1、创建SSH密钥（实现免密登录）：ssh-keygen -t rsa -b 4096 -C "你的邮箱"
	在 C:\Users\你的用户名\.ssh\ 目录下会生成两个文件：id_rsa（私钥，保密）和 id_rsa.pub（公钥）。

	2、在 GitHub 上添加公钥：
		登录 GitHub，点击头像 -> Settings -> SSH and GPG keys -> New SSH key。
		标题任意（如：My Windows PC）。
		用记事本打开刚才生成的 id_rsa.pub 文件，复制全部内容，粘贴到 Key 的文本框中。
		点击 Add SSH Key。

	3、创建项目后使用：
echo "创建项目的名字" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/gaofeng0137/WinGit1.git
git push -u origin main
